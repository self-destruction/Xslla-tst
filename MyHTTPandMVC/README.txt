Ќаписать HTTP приложение на основе MVC парадигмы манипулирующее строками таблицы.
ѕриложение должно использовать фреймворк Silex и механизм автозагрузки PSR-4
ѕриложение должно уметь
вывести всю таблицу  GET /
добавить одну строку POST /
вывести одну строку GET /{id}
обновить одну строку  PUT /{id}
удалить одну строку DELETE /{id}
 {id}  - уникальный идентификатор строки.
